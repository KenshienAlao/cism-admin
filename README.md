# CISM Admin System

This is the central Administrator Portal for managing commercial stalls and vendors. 

The main purpose of this system is to give administrators complete control and visibility over the businesses operating within the facility. Through this dashboard, an admin can:

- **Register Vendors:** Create profiles for new stall owners and automatically generate their secure login credentials (license keys and passwords).
- **Monitor Revenue:** Track exactly how much money each stall is making to see who the top performers are.
- **Oversee Operations:** See which stalls are currently active, view their operating hours, and update their details.
- **Manage Security:** Quickly look up or reset credentials if a vendor loses their login information.

In short, this system is the "command center" that allows the facility manager to oversee all stall operations, finances, and vendor accounts in one place.
